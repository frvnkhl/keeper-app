# Keeper App
## My Google Keep clone

### Stack
- MongoDB database
- Express
- React.js frontend
- Node.js backend

### Dependencies
#### Backend
- body-parser
- cors
- dotenv
- express
- mongoose
- nodemon
- webpack
- webpack-dev-server

#### Frontend
- Material UI
- axios
- firebase
- react
